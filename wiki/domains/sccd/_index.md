# Domain: SCCD (Self · Consciousness · Choice · Decide)

The SCCD-TFAB operator model: a decision framework (functional definitions of Self / Consciousness / Choice / Decide) paired with an accounting logic that makes transparency cheaper than fabrication for a model to produce. TFAB = (contextually) the fabrication-accounting layer this framework attaches to.
